# linuxTools
Random tools and scripts for linux

## Contents of the Directories

### scripts/

Mainly contains bash scripts that I use on various linux distros (e.g., Ubuntu, Fedora, Raspbian)

atlas.sh:          start up Atlas to monitor containers
column.sh:         turn csv file into table
cupsStop.sh:       disable and remove CUPS
fdFind.sh:         remind me how to use fdfind
findGitRemotes.sh: find all of the git remotes below a given directory
findRecentType.sh: find the most recent file of a given type
getReadmes.sh:     find and create symlinks to readme files
ipAddrLocator.sh:  scan for new devices in a given IP address range
localLLM.sh:       query an LLM running on local DGX spark
lsenvs.sh:         tlist names of virtualenvwrapper venvs
maxTemp.sh:        monitor the temperature of a RasPi CPU
mkvenv.sh:         create a new virtualenvwrapper venv
pingAll.sh:        ping all of IPAs in a class C subnet
rssi.sh:           return the RSSI of a local WiFi interface
rsync.sh:          WIP
stream.sh:         stream characters from a tty to the console
tools.sh:          remind me of other tools to use
updateVenv.sh:     update the pip packages in a venv
waypipe.sh:        remotely execute a program on a wayland linux machine
